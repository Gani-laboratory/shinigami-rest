## Example Rest API built with [expressjs](https://expressjs.com/) and written with [typescript](typescriptlang.org/)

### Usage

* Clone the project

```bash
git clone https://github.com/QUERY-GANI/shinigami-rest.git
```

* Installation

```bash
cd shinigami-rest && npm install
```

* Start

```bash
npm start
```
